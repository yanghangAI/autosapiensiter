- `code/pose3d_transformer_head.py` — Added five new kwargs to `Pose3dTransformerHead.__init__` (`kinematic_parametrization`, `bone_parents`, `bone_length_loss_weight`, `per_limb_heads`, `limb_index`), keeping the signature backward-compatible and shared across all three idea013 designs. Added an optional bone-length L1 loss block in `loss()` guarded by `kinematic_parametrization and bone_length_loss_weight > 0.0` (inactive in design001 since the weight is 0.0). `predict()` unchanged.
